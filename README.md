# CountDownly

A single-file countdown timer generator SaaS built with HTML, JavaScript, and Tailwind CSS.

## Features

- **Create Countdown Timers**: Generate custom countdown timers with event names, dates, and colors
- **Embed Code Generation**: Get iframe embed code for easy integration into any website
- **Responsive Design**: Mobile-friendly and works on all devices
- **Stripe Integration Ready**: Includes Stripe Checkout button for subscription ($10/month)

## Usage

1. Open `index.html` in your web browser
2. Enter your event name
3. Set the date and time (format: YYYY-MM-DD HH:MM)
4. Choose a color (hex code)
5. Click "Generate Countdown"
6. Copy the embed code and use it anywhere!

## Technologies

- HTML5
- JavaScript (Vanilla)
- Tailwind CSS (CDN)
- Stripe Checkout (Ready for integration)

## Setup

No installation required! Just open `index.html` in any modern web browser.

For production:
- Deploy the `index.html` file to any static hosting service (GitHub Pages, Netlify, Vercel, etc.)
- For Stripe Checkout, you'll need to implement a backend to create checkout sessions
